# 项目管理系统 - 目录结构

> 版本：1.6.0

## 目录结构

```
project-management/
├── README.md              # 系统概览
├── SKILL.md               # OpenClaw skill 定义
├── INSTALL.md             # 安装指南
├── resource-profiles.md   # 执行资源档案（需填写！）
├── docs/                  # 文档（14 个文件）
│   ├── philosophy.md      # 设计理念
│   ├── coordinator.md     # 派活方手册
│   ├── executor.md        # 执行方手册
│   ├── runtime.md         # 运行时维护
│   ├── decision-guide.md  # 决策指南
│   ├── communication.md   # 沟通协议
│   ├── self-loop.md       # 自循环保障
│   ├── task-management.md # 任务管理
│   ├── quality.md         # 质量保障
│   ├── scheduling.md      # 调度与资源
│   ├── lifecycle.md       # 项目生命周期
│   ├── collaboration.md   # 协作机制
│   ├── knowledge.md       # 知识管理
│   └── governance.md      # 系统治理
├── templates/             # 模板（10 个文件）
│   ├── project-brief.md
│   ├── task-spec.md
│   ├── review-record.md
│   ├── confirmation-record.md
│   ├── milestone-report.md
│   ├── final-report.md
│   ├── change-request.md
│   ├── risk-register.md
│   ├── status.md
│   └── task-registry.md
└── tools/                 # 可选工具
    ├── dashboard.py       # 项目看板
    └── system-check.py    # 系统文件检查
```
