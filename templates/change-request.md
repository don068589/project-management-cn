# 变更请求模板

> 复制本文件到项目的 `changes/` 目录，命名为 `CR[序号]-简述.md`。
> 项目执行中需求变更时填写，经 决策者 批准后执行。

- **变更编号：** CR[序号]

- **优先级：** 高 / 中 / 低
- **状态：** 待评估 / 已批准 / 已拒绝 / 已执行
